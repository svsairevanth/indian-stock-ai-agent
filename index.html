How the Analysis Pipeline Works

A sequential multi-agent pipeline ensures thorough, unbiased evaluation at every stage.

1

Submit Your Query

Enter any NSE/BSE stock symbol or ask a natural language question. Configure your risk profile and investment horizon via the Accuracy Builder.

2

Data Aggregation

The system fetches real-time prices, fundamentals, historical data via yfinance, and live news intelligence through Exa MCP web research.

3

Multi-Agent Analysis

Six specialized agents — Fundamental, Technical, Sentiment, Macro, Document, and News Intelligence — analyze the data in parallel.

Bull and Bear agents construct opposing investment theses. A Debate Judge evaluates both sides and delivers a balanced verdict.

The Risk Manager evaluates portfolio impact. The Orchestrator synthesizes everything into a professional A4 PDF report with BUY/SELL/HOLD recommendation.
